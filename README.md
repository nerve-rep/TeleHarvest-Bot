**TeleHarvest Bot** - Telegram бот на python-telegram-bot и Telethon для выгрузки постов из Telegram каналов. Принимает ссылку на канал и количество постов, возвращает JSON файл с данными. 

**Технологии:** Python, python-telegram-bot, Telethon


ИИ использовался для:
- ускорения разработки компонентов
- написания черновиков функций

Не использовался для:
- архитектуры проекта
- тестирования
